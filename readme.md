# Terraria RPG MOD

A cool RPG mod for Terraria. It adds new rpg classes like swordsman, knight, priest and mage. Each class in TRPG has its own set of buffs and weapons.

### Project structure
```
├───TRPG.cs
├───assets
│   └───32x32
├───Buffs
│   └───Swordsman
├───Items
│   └───Swordsman
├───ModPlayers
└───NPCs
```

### Icons
Icons assets can be found at `./TRPG/assets/32x32` folder.
